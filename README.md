Usage: sfa <-h>    - show help
   or: sfa         - show system info
   or: sfa <c><h>  - also take screenshot(need root)
                      c - console
                      h - homescreen
Example: sfa ch - show system info and take 2 screenshot to your device memory (/sdcard)
